# qlebourgeois.me

This is the http://www.qlebourgeois.me website sources written in SWI-Prolog.

# How to use

* Install SWI-Prolog
* Clone repository
* Enter prolog shell
* Type `consult(bootstrap).`
* Go to http://localhost:5560
